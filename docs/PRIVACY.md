# TermWatch Privacy Policy

TermWatch respects your privacy.

## No Telemetry

TermWatch does not collect, transmit, or store any usage data, analytics, or telemetry.

## No Data Collection

We do not collect personal information, browsing history, or any other user data.

## Local Storage Only

All settings, bookmarks, and preferences are stored locally on your machine using Electron's built-in storage (`electron-store`). Nothing is sent to external servers.

## Update Checks

When checking for updates, TermWatch contacts the GitHub Releases API to determine if a newer version is available. No personal data is included in this request. Update checks can be performed manually from the Settings panel.

## Streaming Content

Video content is loaded directly from streaming services in an embedded browser view. TermWatch does not proxy, record, or intercept this content.
