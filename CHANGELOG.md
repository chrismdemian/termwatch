# Changelog

## 1.0.0

### Features
- Transparent terminal panels overlaid on a web browser video view
- Multiple terminal layouts: 1x1, 1x2, 2x1, 2x2, 1x3
- Per-terminal shell selection (PowerShell, CMD, Git Bash, WSL, zsh, bash, fish)
- DRM support via CastLabs Electron (Widevine) for streaming services
- Video mode with keyboard and overlay navigation controls
- Bookmarks bar for quick site navigation
- Adjustable opacity, shadow intensity, fonts, and color themes
- Persistent sessions, layout preferences, and window state
- Custom frameless window with fullscreen simulation
- Auto-hide controls bar with configurable delay
- OAuth popup handling for streaming service login
- User agent spoofing to bypass bot detection
- Startup autoplay suppression
- Error handling with structured logging and crash recovery
- Auto-update notifications with user-initiated download
- CI pipeline for pull requests and releases
- Third-party license attribution
- NSIS installer with license page (Windows)
- DMG with universal binary (macOS)
